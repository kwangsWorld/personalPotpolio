배열

        JS는 타입이 없으므로,타입 구분없이 데이터를 담을 수 있다.
        크기 지정 필요 없음

1. 선언 및 초기화

2. 메소드


        indexOf() : 배열에서 요소가 위치한 인덱스를 반환함
        concat() : 다수의 배열을 합치고 병합된 배열의 사본을 반환
        join() : 배열 원소 전부를 하나의 문자열로 합친다.
        reverse() :배열의 원소 순서를 거꾸로 바꿈
        sort() : 배열의 원소를 알파벳 순으로, 또는 지정된 함수에 따른 순서로 정렬한다. 모든 원소를 문자열로 취급해 사전적으로 정렬
        push() : 배열 뒷부분 값을 삽입
        pop() : 배열 뒷부분 값을 삭제
        shift() : 배열 앞부분의 값을 삭제
        unshift() :배열 앞부분에 값을 삽입
        slice() : 배열의 startIndex부터 endIndex까지(endIndex는 불포함)에 대한 shallow copy를 새로운 배열 객체로 반환
                  , slice(startIndex, ednIndex)
        splice() : 배열의 특정 위치에 요소를 추가하거나 삭제, splice(index, 제거할 요소 개수, 배열에 추가될 요소)
